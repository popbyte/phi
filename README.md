## Usage

### Prerequisites

* Python
* pip
* uv

### Installation

1. Get a bot token
2. Clone the repo
   ```sh
   git clone --depth 1 https://github.com/popbyte/phi.git
   ```
3. Install dependencies
   ```
   uv pip install --system -r requirements.txt
   ```
4. Run the bot
   ```sh
   python main.py
   ```
